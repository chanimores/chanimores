Projects from Harvard's CS50G

In 2019, I began working on learning how to create 2D games. Specifically, I worked through the 2D projects presented in Harvard's CS50G, Intro to Game Development class that is available online. These projects involve games written in Lua using the LOVE2D game engine.

The projects themselves varied in scope, but given an existing code base, they involved adding features such as adding a powerup to an iteration of the game BreakOut or a pause feature. All in all, I worked through all the 2D projects from the course, and they gave me a great sense of how 2D games might be made in using Lua, learning how collision is calculated manually, how objects can be updated frame-by-frame to move or do animations, and so forth.
