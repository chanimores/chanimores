Meteorite Hackathon Project

This project was originally created by a team for McGill's 2021 Physics Hackathon and later ported onto github. The goal of the project was to create an interactive sphere with NASA meteorite data plotted onto it. Each individual dot on the sphere represents an individual meteorite landing at that corresponding longitude/latitude on the globe.

My role in the project was as one of the programmers. This program was made in Python using the plotly, numpy, and pandas libraries to parse the data and create the plot itself.
