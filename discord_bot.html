Discord Bot Projects

I originally created kermit bot in 2019, and have since created several iterations of the bot with different uses as fun little side projects. The bot is written in JavaScript, using Node.js and discord.js to work.

The first major discord bot project I undertook was a bot that could be used to count up messages. I could call it onto a channel, and the bot would review each message in the channel in batches of 100, counting up the word count in each message and attributing it to a particular user, getting the statistics of each user who spoke in the channel. It was my first foray into understanding how to interact with the discord API and retrieve message info from a particular channel.

The other notable discord bot project I made was an inventory manager bot, which could hold the information of several characters and their items, specifically used for the inventory management of Dungeons & Dragons or other TTRPG player characters. It also had a save/loading system, where it would save all the current characters and their inventories into a JSON file that could be loaded at a later date, if there was a need for multiple casts of characters.
